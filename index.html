Population Shifts in Malaysia: Visualizing Key Trends (1970–2024)

Interactive visualizations depicting the population distribution across Malaysian states. Below you can explore both a choropleth map and a detailed bar chart.

Choropleth Map Description: This map represents the population density of Malaysian states for the year 2020. Each state is shaded according to its population size, with darker colors indicating more densely populated areas. The map provides a clear geographical overview of population distribution, highlighting states like Selangor and Johor as heavily populated regions.
Bar Chart Description: Bar Chart Description: The bar chart provides a comprehensive view of the population distribution across Malaysian states, focusing on data collected every five years. Each bar represents the total population for a specific state in a given year, facilitating straightforward comparisons between states. Users can hover over the bars to see exact population figures, along with details such as the state name and year. The chart includes a selection option to filter the data by individual states or view the combined population data for all states. States like Selangor, Sabah, and Sarawak exhibit significantly larger populations compared to smaller states like Perlis and Labuan.
